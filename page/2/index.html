有效的括号字符串

发表于 2021-09-12 更新于 2021-09-14 分类于 leetcode每日一题

678. 有效的括号字符串

方法1:动态规划

dp表示i到j为有效的字符串。

1.初始 所有*号都是有效的

2.如果满足左右为括号也是有效的

3.状态方程递推

方法2:栈

用两个栈一个存左括号，一个存*号。遇到右括号优先弹出左括号再弹出星号。

最后对两个栈进行遍历。要满足星号变为右括号和左括号进行配对。

最后需要左括号栈为空。

方法3:贪心

正序遍历，把星号当作左括号。确保有足够的左括号进行配对。

逆序遍历，把星号充当右括号。确保有足够的右括号进行配对

用 Rand7() 实现 Rand10()

发表于 2021-09-05 更新于 2021-09-06 分类于 leetcode每日一题

470. 用Rand7()实现Rand10()

面试经典题目。

拒绝采样：生成的随机数满足要求就返回随机数，否则不断生成，直到生成一个满足要求的随机数为止。保证每个生成随机数的概率是相同的。

规律

已知RandM随机生成[1,M]的数字。则（randX() - 1）* Y + randY() 会随机生成[1,X*Y]的数字，且概率相等。

因此通过rand7可以随机生成[1,49]的数字，要使得数字限定在[1,10]之内，则最大的是按照40来取舍，然后取余数。因此不断拒绝，直到得到40之内的数，然后进行取余运算使得范围在rand10。
